# esqueleto-ambulante-webapp

##  Baixar dependências do node
```
yarn install
```

### Copila e aplica hot-reloads para desenvolvimento
```
yarn run serve
```

### Compila e minifica para produção
```
yarn run build
```

### Executa seus testes
```
yarn run test
```

### Lint e corrige arquivos
```
yarn run lint
```
